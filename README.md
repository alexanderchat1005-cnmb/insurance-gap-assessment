## 为什么要改（业务背景）

v1 上线后老板反馈"最好的结果（🛡️ 保险卷王）太多了"。跑了 10 万次蒙特卡洛仿真验证：

- 在真实用户分布下，🛡️ 保险卷王占比 **49.6%**——也就是说一半用户测完被告知"你啥都有，买得还多了"
- 🫣 体检恐惧症患者在所有场景下 **0% 命中**，是死档人格
- 这个测评挂在集团保险中介平台下，目的是引导用户发现缺口→咨询/购买保险，卷王一多，转化必崩

v2 改版的验收结果（仿真 10 万次/场景）：

| 场景 | v1 保险卷王 | v2 保险卷王 | v1 体检恐惧 | v2 体检恐惧 |
|---|---:|---:|---:|---:|
| Uniform (1:1:1) | 10.9% | **0.8%** | 0.0% 💀 | **3.3%** |
| **C-biased (20/35/45)** ← 真实用户分布 | **49.6%** | **13.3%** | 0.0% 💀 | 2.2% |
| Strong-C (15/30/55) | 80.4% | **41.6%** | 0.0% 💀 | 0.8% |
| A-biased (45/35/20) | 0.4% | 0.0% | 0.0% 💀 | 1.8% |

核心 KPI：C 偏好场景下，保险卷王占比 **49.6% → 13.3%**（降 73%）；体检恐惧症从死档救活。

详细的题目增量清单、分值变化、人格 pattern 变化，见 [`.marketing/改版说明.md`](.marketing/改版说明.md)。
